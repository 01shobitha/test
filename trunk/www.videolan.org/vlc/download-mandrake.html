    Mandrake 9.1 and Cooker

    To install the latest VLC packages on either Mandrake 9.1 or Cooker, configure the package sources (you can use Easy urpmi for that) with at least the following sources:

    • from the core distribution: main and contrib;
    • in the external add-ons: plf (Penguin Liberation Front).

    Eventually, install the required packages with urpmi:

       # urpmi libdvdcss2 libdvdplay0 wxvlc vlc-plugin-a52 vlc-plugin-ogg vlc-plugin-mad
