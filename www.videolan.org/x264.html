x264 - a free h264/avc encoder

x264 is a free library for encoding H264/AVC video streams. The code is written by Laurent Aimar, Eric Petit(OS X), Min Chen (vfw/nasm), Justin Clay(vfw) and Måns Rullgård from scratch. It is released under the terms of the GPL license.

Development status

BIG FAT WARNING: x264 is still in early development stage

Encoder features

  • CAVLC/CABAC
  • Multi-references
  • Intra: all modes (4x4 and 16x16 with all predictions
  • Inter P: all partitions from 16x16 downto 4x4
  • Inter B: 16x16 modes only (and no SKIP/DIRECT). They aren't yet usable as they don't improve the coding efficiency
  • Constant bitrate mode
  • Scene cut detection

Encoder limitations

  • No real RD

Getting x264

The latest x264 source code can always be found by anonymous SVN repository:

# svn co svn://svn.videolan.org/x264/trunk x264
