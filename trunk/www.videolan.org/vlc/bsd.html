vlc on BSD (FreeBSD, OpenBSD, NetBSD, BSD/OS)

If you are using vlc on FreeBSD, OpenBSD, NetBSD or BSD/OS, have a regular look at this page to see how the port is going along.

Current status

Program compiling: yes
Additional requirements: none
Plugin support: yes
CPU-specific acceleration: MMX, MMX EXT, SSE, 3D Now! *
Native interface: Gnome, Gtk+, Qt
Native video output: X11, SDL
YUV acceleration in video output: X11, SDL
Native audio output: OSS, Esound
File input: yes
Unencrypted DVD input: yes
Encrypted DVD input: yes
VCD input: BSD/OS, FreeBSD
MPEG-4/DivX support: yes
IPv4 Network input: yes
IPv6 Network input: untested
HTTP protocol : yes
Satellite card : no

Additional information

* = OpenBSD 2.9 default assembler doesn't support MMX

The x86 *BSD are very close to GNU/Linux, our main development platform, so backporting Linux changes to *BSD will be quite straightforward.
