    VLC for Linux

    Download options

    You have two choices :

    • download a set of packages for your architecture / distribution. You can choose the set of plugins that you'd like to use with vlc and download their individual packages. This option is the easiest one, but some exotic configurations may not have a binary package yet.
    • download a source tarball of vlc and compile it yourself. This will work in all cases, but you'll need additional packages and you should be very familiar with the build process of your architecture.

    Source package:

    Since there are many RPM-based distributions, you may get a better package by rebuilding it yourself. Download the SRPM and run for instance rpm --rebuild vlc-0.4.0-1.src.rpm. The RPMs can then be found in /usr/src/RPM/RPMS/.
