OpenSource Video streaming solution for every OS !

VideoLAN is a project of French students from the École Centrale Paris and developers from all over the world. Its main goals is MPEG streaming on a network, but it also features a standalone multimedia player. The VideoLAN Server can stream MPEG videos read from a hard disk, a DVD drive, a satellite card, a digital terrestial television card or an MPEG 2 compression card, and unicast or multicast it on a network. The VideoLAN Client can read the stream from the network and display it. It can also be used to display video read locally on the computer : DVDs, VCDs, MPEG and DivX files and from a satellite card. It is multi-plaform : Windows, Mac OS X, Linux, BeOS, BSD, Solaris, QNX, Familiar Linux... The VideoLAN Client and Server now have a full IPv6 support.

VideoLAN is free software, and is released under the GNU General Public License.

More information about the VideoLAN streaming solution be found in the streaming overview page. You can also check the mailing-lists, and if you want more real-time stuff, you can meet the VideoLAN developers in #videolan on the freenode network (former OpenProjects network) (irc.freenode.net).

Contribute !

VideoLAN welcomes all contributions to the project ! You can contribute time (developpement, documentation, tests, user support, ...), material or even money. See the contribute page for more information.
