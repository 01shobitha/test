6. Step 5 : Multicast two streams to vlc(s) with a mini-vlcs

You don't have to change anything to vls. You are going to install the mini VideoLAN Channel Server (mini-vlcs) and configure-it. Then, vlc(s) will be able to change channels very easily via the graphical interface. You can install mini-vlcs on the same machine as vls.

6.1 Install mini-vlcs

Download the last version of miniVLCS from the Network download page.

Install-it : 

% tar xvzf miniVLCS-version.tar.gz
% cd miniVLCS-version
% make

6.2 Configure mini-vlcs

Edit the configuration file vlcs.conf.

The first line corresponds to channel 0, the second line corresponds to channel 1, the third line to channel 2, and so on.

In each line, write what you would type on the command line of vlc, except for the first line I: nostream. This mean that you don't stream anything in channel 0.

In our example, the configuration file vlcs.conf should look like that :

I: nostream
udp:@239.255.12.42
udp:@239.255.12.43

6.3 Run mini-vlcs

As normal user, type : 

% ./vlcs

For production, you should run the program in a screen, so that you can detach-it and reattach-it : 

% screen ./vlcs

6.4 Run vlc(s)

The simplest solution is to use the interface of vlc : click on "Net", select "Channel server", write its IP address or DNS name.

Then, you see a new section "Network Channel" on the interface. Select "1" and click on "Go!". You should see the presentation. Then, select "2" and click on "Go!". You should see the DVD. Enjoy !
