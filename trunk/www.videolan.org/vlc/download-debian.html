    • VLC for Debian GNU/Linux

    Add the following lines to your /etc/apt/sources.list:

       deb     ftp://ftp.videolan.org/pub/videolan/debian $(ARCH)/
   deb-src ftp://ftp.videolan.org/pub/videolan/debian sources/

    For a normal install, do:

       # apt-get update
   # apt-get install gnome-vlc libdvdcss2

    Note : the last version of VLC is now part of the unstable distribution (sid). You should still use the procedure above if you want to have the lastest version of VLC and libdvdcss.
