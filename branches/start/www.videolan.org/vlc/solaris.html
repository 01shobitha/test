vlc on Solaris

If you are using vlc on Solaris, have a regular look at this page to see how the port is going along.

Current status

Program compiling: yes
Additional requirements: none
Plugin support: yes
CPU-specific acceleration: x86 only (MMX)
Native interface: Gnome, Gtk+, Qt
Native video output: X11, SDL
YUV acceleration in video output: untested
Native audio output: no
File input: yes
Unencrypted DVD input: yes
Encrypted DVD input: yes
VCD input: no
MPEG-4/DivX support: yes
IPv4 Network input: yes
IPv6 Network input: untested
HTTP protocol : yes
Satellite card : no

Known issues

(none)
