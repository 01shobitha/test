    Install VLC with urpmi

    There are VLC packages for Mandrake 9.1 and cooker.

    To install them, configure the package sources with Easy urpmi (you need at least to add main, contrib and plf sources).

    If you are running cooker, you don't need to configure anything else, as VLC packages are part of the contrib section.

    If you are running Mandrake 9.1, add the VideoLAN package source:

    urpmi.addmedia vlc http://www.videolan.org/pub/videolan/vlc/0.6.0/rpm/mdk/

    Eventually, install the packages with urpmi:

    urpmi libdvdcss2 libdvdplay0 wxvlc vlc-plugin-a52 vlc-plugin-ogg vlc-plugin-mad
