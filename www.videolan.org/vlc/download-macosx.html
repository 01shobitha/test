    Latest Mac OS X package (0.7.1a)

    Note: version 0.7.1a fixes two small issues in 0.7.1. Make sure you have the latest version.

    Note: this version does not support Mac OS 9. There is not, and won't be, a version for Mac OS 9.

    VLC for Mac OS X is distributed as a disk image. Download the .dmg file, open it, and copy it to your hard drive (for instance the /Applications folder). Open VLC by double-clicking on it.
