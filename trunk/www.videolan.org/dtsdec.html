dtsdec/libdts - a free DTS Coherent Acoustics decoder

libdts is a free library for decoding DTS Coherent Acoustics streams. The code is written by Gildas Bazin and was based on the a52dec project. It is released under the terms of the GPL license.

The DTS Coherent Acoustics standard (ETSI 102 114 v1.2.1), as published by the ETSI, is available here (look for DTS Coherent Acoustics).

Development status

BIG FAT WARNING: libdts is still in early development stage so don't expect too much of it

Right now, I am concentrating on finishing the implementation of the core part or the specification.

A few things are still lacking there but they should mainly affect low-bitrate DTS streams which don't seem that common anyway (some of the missing things are: Joint channels coding, High frequencies VQ, etc...).

There might also be quality issues, but for some early code it does seem pretty decent to me.

Getting libdts

Current release is dtsdec-0.0.1.tar.gz

The latest libdts and dtsdec source code can always be found by anonymous SVN repository:

# svn co svn://svn.zoy.org/dtsdec
