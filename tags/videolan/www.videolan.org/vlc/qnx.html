vlc on QNX RTOS

If you are using vlc on QNX RTOS, have a regular look at this page to see how the port is going along.

Current status

Program compiling: yes
Additional requirements: none
Plugin support: yes
CPU-specific acceleration: MMX
Native interface: no
Native video output: yes (X11 emulation also)
YUV acceleration in video output: yes
Native audio output: yes
File input: yes
Unencrypted DVD input: untested
Encrypted DVD input: no
VCD input: no
MPEG-4/DivX support: untested
IPv4 network input: no
IPv6 network input: no
HTTP protocol : yes
Satellite card : no

Known issues

The native video output is pretty slow.
