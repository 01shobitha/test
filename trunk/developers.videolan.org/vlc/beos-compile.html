Compile VLC on BeOS

Needed tools:

  • You need BeOS 5 or higher with developpement tools.
  • You need a few utilities for the build process, and several librairies in order to make a full VLC build for BeOS. You can build them yourself, or you can download and untar to /boot/home/config/ this archive (last updated: 2003/05/12), which contains the following compiled for BeOS:
    • cvs 1.11
    • perl 5.8.0
    • autoconf 2.57
    • automake 1.7.3
    • libtool 1.4.2
    • gettext 0.11.5 w/ libiconv 1.8
    • a52dec 0.7.4
    • ffmpeg CVS-20030510
    • libdvdplay 1.0.1
    • libdvdread 0.9.4
    • libdvdcss 1.2.6
    • faad2 CVS-20030505
    • FLAC 1.1.0
    • mpeg2dec CVS-20030505
    • libogg 1.0
    • libvorbis 1.0
    • mad 0.14.2b
  • In order to use these libraries, add the following lines to your /boot/home/.profile:
    export C_INCLUDE_PATH="${C_INCLUDE_PATH}:/boot/home/config/include"
    export CPLUS_INCLUDE_PATH="${CPLUS_INCLUDE_PATH}:/boot/home/config/include"
    export LDFLAGS="${LDFLAGS} -L/boot/home/config/lib"

Now you can compile vlc:

  • download the source code (use CVS or download the archive)
  • cd to the source directory
  • if you are using CVS, run ./bootstrap
  • run ./configure. The following is suggested:
    ./configure --enable-debug --disable-plugins --enable-ffmpeg --enable-flac --enable-faad
  • run make
  • to make a package, run make package-beos
