    Thanks to Götz Waschk who made these packages !

    Mandrake 9.1

    To install the VLC packages, configure the package sources with Easy urpmi (you need at least to add main, contrib and plf sources).

    Then add the VideoLAN package source:

       # urpmi.addmedia vlc http://www.videolan.org/pub/videolan/vlc/0.6.2/rpm/mandrake-9.1/ with hdlist.cz

    Eventually, install the packages with urpmi:

       # urpmi libdvdcss2 libdvdplay0 wxvlc vlc-plugin-a52 vlc-plugin-ogg vlc-plugin-mad

    Mandrake Cooker

    To install the VLC packages, configure the package sources with Easy urpmi (you need at least to add main, contrib and plf sources).

    VLC packages are part of the contrib section; so you can install the packages directly with urpmi:

       # urpmi libdvdcss2 libdvdplay0 wxvlc vlc-plugin-a52 vlc-plugin-ogg vlc-plugin-mad
