    Install VLC from the RPM packages

    Download the RPM package vlc and the packages listed in the required libraries and codecs section below (the other packages are just optional) and put them all into the same directory.

    Then install the packages:

        # rpm -U *.rpm

    If you have not installed all the RPM packages included with your distribution, you may be asked to install a few of them first.
