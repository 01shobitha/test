VLC media player Mac OS X Intel nightly builds

Remember to test the latest nightly build before reporting a bug (in one of the older nightly builds). Have fun!

Please report nightly builds breakage (meaning that there is no new build) on #videolan on irc.freenode.net .

Note: VLC requires Mac OS X 10.5 or later

It will not launch on older releases. We recommend to use the latest version available for your release.


vlc-git-intel is the latest VLC snapshot of the unstable branch, compiled (32 bit)

1.1-branch-intel is the latest VLC snapshot of the stable branch (32 bit)

1.1-branch-intel64 is the latest VLC snapshot of the stable branch (64 bit)

vlckit-git-intel contains an Objective-C framework to create your own applications based upon VLC.
