VLC for Gentoo Linux

Install VLC via normal emerge

First, VLC is included in the Portage package system of the Gentoo Linux distribution. So you can simple type emerge vlc and you will have VLC installed. However this version ( currently 0.4.4) is very outdated, so I advice to use ACCEPT_KEYWORDS="~x86" emerge vlc to emerge the current unstable version of VLC.

Get the latest and greatest and heavily unstable stuff

From version 0.6.0 and up every release (and if you are lucky even cvs snapshots) come with there own ebuild. This is to help the Gentoo Developers with keeping an up to date ebuild in portage (with all the latest and greatest features). However, this will also help you to get the latest version of VLC running on your Gentoo system within minutes after a new release if you want. ( portage can't keep up with your desire to run the latest VLC and you know it ;).

See the VLC ebuild tutorial for more information.
