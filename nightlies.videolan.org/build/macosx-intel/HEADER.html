VLC media player Mac OS X Intel nightly builds

Remember to test the latest nightly build before reporting a bug (in one of the older nightly builds). Have fun!

Please report nightly builds breakage (meaning that there is no new build) on #videolan on irc.freenode.net .

Note: VLC's nightly builds for Intel require Mac OS X 10.6 or later

It will not launch on older releases. We recommend you to keep your Mac OS X installation up-to-date and to install Apple's updates. This will also improve your VLC usage experience.
VLC's official releases still support Mac OS X 10.5.


What's the stable branch? That's currently VLC's 1.2 series. We no longer provide nightly builds for VLC 1.1.x.


vlc-i686 is the latest VLC snapshot of the unstable branch, compiled in 32 bit mode

vlc-x86_64 is the latest VLC snapshot of the unstable branch, compiled in 64 bit mode

vlc-stable-i686 is the latest VLC snapshot of the stable branch (32 bit)

vlc-stable-x86_64 is the latest VLC snapshot of the stable branch (64 bit)
