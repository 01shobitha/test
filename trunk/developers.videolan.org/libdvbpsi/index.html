libdvbpsi

libdvbpsi is a simple library designed for MPEG TS and DVB PSI tables decoding and generation. The important features are:

  • Portability. Currently supported platforms are GNU/Linux, Windows and Mac OS X. It should work on a lot of platforms since the code is totally portable. Test and report on failure or success.
  • Freedom. libdvbpsi is released under the General Public License, ensuring it will stay free, and used only for free software products.

Current support

  • Program Association Table (PAT), decoder and generator.
  • Program Map Table (PMT), decoder and generator.
  • All MPEG 2 descriptors, decoders and generators.
