Compile VLC on BeOS

Needed tools:

  • You need BeOS 5 or higher with developpement tools.
  • You need a few utilities for the build process:
    • perl 5.8.0
    • autoconf 2.57
    • automake 1.7.3
    • libtool 1.4.2
    • gettext 0.11.5
    This archive (last updated: 2003/03/25) includes all these tools built for BeOS. Just download it and untar it to /boot/home/config/.
  • You need several librairies in order to make a full VLC build for BeOS:
    • a52dec 0.7.4
    • libavcodec CVS-20030304
    • libdvdplay 1.0.1
    • libdvdread 0.9.4
    • libdvdcss 1.2.6
    • libfaad CVS-20030130
    • libFLAC 1.1.0
    • libmpeg2 CVS-20030325
    • libogg 1.0
    • libvorbis 1.0
    • mad 0.14.2b
    This archive (last updated: 2003/03/25) includes all these libraries built for BeOS. Just download it and untar it to /boot/home/config/.
  • In order to use these libraries, add the following lines to your /boot/home/.profile:
    export CFLAGS=-I/boot/home/config/include
    export CPPFLAGS=-I/boot/home/config/include
    export CXXFLAGS=-I/boot/home/config/include
    export LDFLAGS=-L/boot/home/config/lib
  • Convenient: cvs.

Now you can compile vlc:

  • Download the source code (use cvs or download the archive)
  • cd to the source directory
  • run ./bootstrap
  • run ./configure. The following is suggested:
    ./configure --enable-debug --enable-a52 --enable-ffmpeg --enable-dvdplay --enable-faad --enable-mad --enable-ogg --enable-vorbis --enable-libmpeg2 --enable-flac
  • run make
