vlc on BeOS

If you are using vlc on BeOS, have a regular look at this page to see how the port is going along.

Current status

Program compiling: yes
Additional requirements: none
Plugin support: yes
CPU-specific acceleration: MMX
Native interface: yes
Native video output: yes
YUV acceleration in video output: yes
Native audio output: yes
File input: yes
Unencrypted DVD input: yes
Encrypted DVD input: yes
VCD input: no
MPEG-4/DivX support: yes
IPv4 Network input: yes (with bone)
IPv6 Network input: no
HTTP protocol : yes
Satellite card : no

Known issues

For some yet unknown reason, DVD decryption can be very slow as we try to crack the disc key. It can sometimes take up to 10 minutes. This problem should be solved when we find a better way to crack the key.
