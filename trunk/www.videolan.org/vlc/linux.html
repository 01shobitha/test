vlc on Linux

If you are using vlc on Linux, have a regular look at this page to see how the port is going along.

Current status

Program compiling: yes
Additional requirements: none
Plugin support: yes
CPU-specific acceleration: MMX, MMX EXT, SSE, 3D Now!, Altivec
Native interface: Gnome, Gtk+, Qt
Native video output: X11, SDL, framebuffer, GGI
YUV acceleration in video output: X11, SDL
Native audio output: OSS, Esound, ALSA
File input: yes
Unencrypted DVD input: yes
Encrypted DVD input: yes
VCD input: yes
MPEG-4/DivX support: yes
IPv4 Network input: yes
IPv6 Network input: yes
HTTP protocol : yes
Satellite card : yes

Additional information

GNU/Linux is our main development platform, so the Linux port will always be the most complete.
