Skins

VLC has now a skinnable interface. It's only available on Windows systems and is experimental. That means it should work on Windows 2000 and has not yet been tested on Win9x/WinXP OS.

Actually we are working on a GTK version so that this feature will be soon available on Linux platforms.

You can test the skinnable interface in the 0.5.3 release. There is a shortcut in the start menu can launch this interface.

Or you can launch it with command line:

vlc.exe -I skins

Or go to the preference menu after launching the client and ckick on the View/Add Interface/skins menu.

You need a skin file too, you can download them. These are just testing skins but if you think you're some kind of artist, you don't need any programming skills so here is how to create your skin.

Not all VLC features are available with skins plugin but here are what it supports :

  • Playing a file
  • Seeking into a stream
  • Sound control
  • Playlist
  • Key shortcuts
