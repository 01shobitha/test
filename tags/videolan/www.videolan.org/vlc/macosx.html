VLC on MacOS X

VLC currently only works on Mac OS X 10.1 and later. There is not, and will never be, a version of VLC for Mac OS 9, so please don't ask !

VLC - OS X is distributed as a disk image. Download the .dmg file, open it, and copy it to your hard drive (for instance the /Applications folder). Open VLC by double-clicking it.

Hackers wanted ! If you feel you can help on the port, don't hesitate to send us a mail. Please do not send us localized versions of VLC. The next version will have localization support, through a scheme which isn't compatible with Apple's, so please don't waste your time...

Building VLC

All the building process has to be done from a terminal window. If you are familiar with Unix, this won't be a problem for you.

After downloading vlc, just type ./configure && make to build the main vlc application.

Starting from VLC 0.2.90, a binary distribution is also available is the download section.
