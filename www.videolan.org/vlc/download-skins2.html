Skins VS skins2

Important note:
VLC currently has two different skins modules, named skins and skins2, the latter being a complete rewrite of the former. We consider that the skins module is deprecated, and we strongly encourage people to use skins2 when creating a new skin. The skins module is only kept because there are still very few skins ported to the new skins2 module, but it is going to disappear one day or other...

In this page you will only find skins for the new skins2 module. There is a different download page for the old deprecated skins module.

How to use these skins?

See this page for more details on how to load a downloaded skin.

Note for Windows users:
In versions of VLC up to 0.7.2 (included), a bug prevents to load the .vlt file as indicated (it is fixed in newer versions). Here is a workaround:

  • Rename the .vlt file in a .tar.gz file (ex: winamp5.vlt becomes winamp5.tar.gz)
  • Extract the files from the .tar.gz archive using your favourite compression software
  • When changing the skin, specify the theme.xml file that you just extracted (among other files) instead of the .vlt file
  • That's it!
