VLC for Familiar Linux

A feed for Familiar Linux version 0.5 and 0.6 is available at the URL http://www.videolan.org/pub/videolan/vlc/ipkg-feed/armv4l to get VLC easily. There are packages available for Familiar Linux Gtk+ and GPE.

Packages upto VLC version 0.4.4 support Familiar Linux version 0.5.
Packages from VLC version 0.4.5 onwards support Familiar Linux version 0.6.

If upgrading from vlc (vlc, gpe-vlc and opie-vlc) version 0.5.0 or from any test packages leading up to version 0.5.1, then you'll have to uninstall the additional package called ffmpeg. The package is renamed to libffmpeg, because it only provides libraries from the ffmpeg tree.

  • Uninstall ffmpeg with: ipkg remove ffmpeg
  • Install libffmpeg with: ipkg install libffmpeg_0.4.6_20030211_arm.ipk

Latest VLC packages for Familiar Linux 0.6 and above:

Interface and Codecs (they come in the same package):

  • Gtk+ interface: vlc_0.5.3_arm.ipk
  • GPE interface: gpe-vlc_0.5.3_arm.ipk

Required library:

  • libffmpeg_0.4.6_20030304_arm.ipk

Latest VLC package for Familiar Opie is 0.5.3:

Interface:

  • Commandline interface: opie-vlc_0.5.3_arm.ipk

Required libraries:

  • libffmpeg_0.4.6_20030304_arm.ipk
  • opie-libsdl_1.2.5_arm.ipk
