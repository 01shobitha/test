Compile VLC on BeOS

Required tools:

  • BeOS 5 with development tools
  • CVS 1.11
  • Subversion
  • bzip2/bunzip2
  • wget
  • gcc 2.95.3

Compiling is usually done on a vanilla R5.0.3 install with gcc 2.95.3. Building with older compilers won't work. Building on a BONE-enabled install probably doesn't work at the moment.

Now you can compile vlc:

  • download the source code using Subversion
  • cd to extras/contrib/
  • run ./bootstrap && make src - this will get and build every util and library VLC needs (you need an internet connection, a fast one preferably)
  • cd to the root of the source directory
  • run ./bootstrap && ./configure --enable-debug --enable-sout --enable-httpd --enable-vlm --enable-dvdread --enable-dvdnav --enable-dvbpsi --enable-screen --enable-ogg --enable-mkv --enable-mad --enable-ffmpeg --enable-faad --enable-a52 --enable-flac --enable-libmpeg2 --enable-vorbis --enable-speex --enable-theora --enable-freetype --enable-fribidi --disable-skins2 --with-ffmpeg-mp3lame --with-ffmpeg-faac --with-ffmpeg-zlib && make
  • to build a package, run make package-beos
