VLC for Familiar Linux

A feed for Familiar Linux version 0.5 and 0.6 at the internet location http://www.videolan.org/pub/videolan/vlc/ipkg-feed/armv4l is available for getting the VideoLAN client. There are packages available for Familiar Linux Gtk+ and GPE.
Packages upto vlc version 0.4.4 support familiar version 0.5.
Packages from vlc version 0.4.5 onwards support familiar version 0.6.

Latest vlc versions for Familiar Linux v0.6 are:

Interface and Codecs

The interface and codecs come in the same package.
  • Gtk+ interface: vlc_0.5.0_arm.ipk
  • GPE interface: gpe-vlc_0.5.0_arm.ipk

Libraries

Additional libraries needed:
  • ffmpeg_0.4.6_20030201_arm.ipk

    • Latest version for Familiar Opie is 0.4.6

    Interface

    • Commandline interface:opie-vlc_0.4.6_pre2_arm.ipk
