How can you help

We always need new translators to help with localization of VLC. You can create a localization for an unsupported langage, help the translators of one given langage (either by translating or by reviewing)

For each existing langage, there is a translation maintainer. You should contact him to coordinate help.

If you want to make a translation for a new langage, please read the howto
