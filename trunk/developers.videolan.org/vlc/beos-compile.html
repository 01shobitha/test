Compile VLC on BeOS

Needed tools:

  • BeOS 5 or higher with developpement tools.
  • Perl 5.8.0.
  • CVS 1.11.
  • This archive includes the following tools, which are needed for the build process. Unzip it to /boot/home/config/.
    • autoconf 2.57
    • automake 1.7.5
    • libtool 1.5
    • libiconv 1.9.1
    • gettext 0.11.5
  • This archive includes the following libraries, which are needed for the build process. Unzip it to /boot/home/config/.
    • a52dec 0.7.4
    • libavcodec CVS-20030612
    • libdvdplay 1.0.1
    • libdvdread 0.9.4
    • libdvdcss 1.2.6
    • libfaad CVS-20030612
    • libFLAC 1.1.0
    • libmpeg2 CVS-20030612
    • libogg 1.0
    • libvorbis 1.0
    • libmad 0.14.2b
  • In order to use these libraries, add the following lines to your /boot/home/.profile:
    export C_INCLUDE_PATH="${C_INCLUDE_PATH}:/boot/home/config/include"
    export CPLUS_INCLUDE_PATH="${CPLUS_INCLUDE_PATH}:/boot/home/config/include"
    export LDFLAGS="${LDFLAGS} -L/boot/home/config/lib"

Now you can compile vlc:

  • download the source code (use CVS or download the archive)
  • cd to the source directory
  • if you are using CVS, run ./bootstrap
  • run ./configure. The following is suggested:
    ./configure --enable-debug --disable-plugins --enable-flac --enable-faad
  • run make
  • to make a package, run make package-beos
