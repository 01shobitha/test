VLC media player nightly builds

Welcome!

Here you can download experimental versions of VLC that are daily generated from the VLC code that is currently in development. Several issues that existed in previous versions of VLC might be fixed here, but since this version of VLC is not yet finished, you should expect other problems and issues.

Download nightly builds

You can find now versions from branch 0.8.6-bugfix and versions from the trunk (future 0.9.0). You can see the differences with their names (branch-xxx against trunk-xxx) in the same directories.

Select your operating system to download VLC binaries:

  • Windows
  • Mac OS X PowerPC
  • Mac OS X Intel
  • BeOS
  • Debian Sid I386

  • Ubuntu Breezy I386 (stable) snapshot

Source-code snapshots

Download here the tarballs (bz2) or use SVN to get the more recent trunk.

Debian and Ubuntu repositiories

For Debian Sid I386 (unstable) add the following line to your sources.list:

deb http://nightlies.videolan.org/build/sid-i386/arch ./

For Debian Sid AMD64 (unstable) add the following line to your sources.list:

deb http://nightlies.videolan.org/build/sid-amd64/arch ./

Those repositories contain a vlc-dbg package that you can install so you can run gdb to provide usefull backtrace when reporting bugs.

Our debian archives are signed using this key (AA5F4DE6). To use it, either:

  • Retrieve it directly (but you don't really check the key):
    As root, run : 
    wget -O /dev/stdout http://nightlies.videolan.org/key|apt-key add -
  • Import it in your gnupg installation, either from the file or from any keyserver, and check it (it is signed by several VideoLAN developers). 
    gpg --keyserver subkeys.pgp.net --recv-keys AA5F4DE6
gpg --list-sigs AA5F4DE6 (check for errors)
gpg --export AA5F4DE6 -a|sudo apt-key add -

For Ubuntu Breezy I386 (stable) snapshot add the following line to your sources.list:

deb http://nightlies.videolan.org/build/breezy-i386 /

This package is not automatically built every night. Our breezy archive is signed using this key (AA5F4DE6). To use it, you may follow the same steps as for our debian archive's key.

For Ubuntu Dapper I386 (dapper) snapshot add the following line to your sources.list:

deb http://nightlies.videolan.org/build/dapper-i386 /

This package is not automatically built every night. Our breezy archive is signed using this key (AA5F4DE6). To use it, you may follow the same steps as for our debian archive's key.
