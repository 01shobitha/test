VLC - a multi-platform media player

Overview

The VLC (VideoLAN Client) media player runs on many platforms : Linux, Windows, Mac OS X, BeOS, *BSD, Solaris, Familiar Linux and QNX. It can read MPEG 1, MPEG 2 and MPEG 4 / DivX files from a hard disk or a CD-ROM drive, read DVDs and VCDs and read from a satellite card. But the most important feature is that it can read MPEG 1, MPEG 2 and DVDs streamed over a network.
