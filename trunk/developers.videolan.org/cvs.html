CVS repository

Basic information

The anonymous CVS server is called anoncvs.videolan.org. You can log in as user anonymous, with an empty password. For instance, to retrieve the vlc source: 

cvs -d :pserver:anonymous@anoncvs.videolan.org:/var/cvs/videolan login
cvs -d :pserver:anonymous@anoncvs.videolan.org:/var/cvs/videolan -z3 checkout -P vlc

In case you do not have access to CVS (due to strange corporate policies), CVS tarball snapshots are available (see below).

Available modules

  • vlc: VideoLAN Client;
  • vls: VideoLAN Server;
  • vlms: VideoLAN MiniServer;
  • network: network tools (VLAN-Server and VLAN-Bridge).
  • libdvdcss: library for decrypting DVDs, used by VLC and VLS.
  • libdvdplay: library for DVD navigation, used by VLC.
  • libdvbpsi: library for parsing Program Specific Information from MPEG and DVB streams, used by VLC and VLS.

CVSWeb

You can browse the CVS repository using a nifty tool called CVSWeb.

VLC

There is only the MAIN branch at the moment. It is the one you get by default with CVS.

VLS

VLS is currently split into a stable branch and an unstable branch. The unstable branch is the branch you get by default with CVS, and the stable branch is a bug-fix-only branch. Current stable branch is tagged v0_5_0 and 0.5.x versions of VLS come from it. To access it : 

cvs -z 3 -d :pserver:anonymous@cvs.videolan.org:/cvs/videolan checkout -P -r v0_5_0 vls

The unstable branch is under active development and may sometimes be broken. Patches for new features should be sent against it.

CVS tarball snapshots

If you have trouble using CVS, you can still use our daily VLC CVS snapshots.

Write access policy

As usual in opensource projects, you get write access to the CVS repository when you have shown your skills and ability to write good patches :)
