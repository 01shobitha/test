libdvdplay

libdvdplay is a simple library designed for DVD navigation. It is based on libdvdread and optionally libdvdcss

Currently supported platforms are Unix, Windows, BeOS and Mac OS X, but it should work on almost any platform.

Get libdvdplay

libdvdplay is available through CVS or by browsing the releases.

You can also see the release notes.
