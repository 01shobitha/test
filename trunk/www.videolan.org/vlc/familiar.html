VLC on iPAQ Familiar Linux

If you are using VLC on iPAQ, have a regular look at this page to see how the port is going along. The Familiar Linux port is now working well, with full audio, video and network support.

Hackers wanted ! If you feel you can help on the port, don't hesitate! In particular StrongARM asm could be written to optimize the YUV transform.

Current status

Program compiling: yes
Additional requirements: SDL recommended
Plugin support: yes
CPU-specific acceleration: none
Native interface: no
Native video output: x11, SDL
YUV acceleration in video output: no hardware support
Native audio output: yes
File input: yes
Unencrypted DVD input: no hardware support
Encrypted DVD input: no hardware support
VCD input: no hardware support
MPEG-4/DivX support: yes but slow...
IPv4 network input: yes
IPv6 network input: yes
HTTP protocol : yes
Satellite card : no

Getting VLC

We have set up a feed for Familiar Linux 0.5. For your ipkg configuration : ftp://ftp.videolan.org/pub/videolan/vlc/ipkg-feed/armv4l
