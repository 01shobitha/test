Another great start for documentation is the source code. The best efforts have been done to make it as understandable as possible. You can also ask questions in the development mailing-list vlc-devel@videolan.org, or try to meet the developers on IRC (#videolan, irc.videolan.org or any OPN server) if you want to help us but don't know where to start.
