2. Step 1 : Install vlc and read streams locally

2.1 Install vlc

under Windows

vlc works under Windows 95/98/ME/2000/XP. You can download a Windows Installer or a Zip file from the vlc download page. Execute the Windows installer or unzip the file in a directory to install vlc.

under BeOS

You can download a Zip file from the vlc download page. Unzip the file in a directory to install vlc.

under Mac OS X

Download the Mac OS X package from the vlc download page and double-click on the icon of the package.

under Linux Debian

Edit /etc/apt/sources.list and add the following line :

deb http://www.videolan.org/pub/videolan/debian $(ARCH)/
deb-src http://www.videolan.org/pub/videolan/debian sources/

Then type as root :

# apt-get update
# apt-get install vlc-gnome vlc-mad libdvdcss2

under Linux RedHat, Mandrake or SuSE

Install liba52 and libmad0. They should be packaged in your distribution. If they are not, try to find RPMs on the Internet.

Download the RPM package of libdvdcss from libdvdcss download page.

Download the RPM package of libdvbpsi from libdvbpsi download page.

Download the RPM packages vlc vlc-gnome and vlc-mad from the vlc download page.

Install the packages. As root, type :

# rpm -Uhv vlc-versionmdk.i586.rpm vlc-mad-versionmdk.i586.rpm vlc-gnome-versionmdk.i586.rpm libdvdcss2-version.i586.rpm libdvbpsi1-version.i586.rpm

under Unix or Linux

The method below is for any Unix or Linux supported by vlc. We are going to compile and install the librairies and then vlc.

Install the librairies

Many librairies are needed for particular uses :

  • libdvdcss if you want to read encrypted DVDs,
  • libdvbpsi if you want to read from the network (not compulsory for the moment),
  • a52dec if you want to decode AC3,
  • ffmpeg and libmadif you want to read MPEG 4 / DivX files.

Download the librairies from the VideoLAN FTP site, ftp://ftp.videolan.org/pub/videolan/vlc, in the directory version/contrib.

For each librairie, uncompress, configure, compile and install : 

% tar -xvzf librairie.tar.gz
% cd librairie
% ./configure
% make
% su
 [enter root password]
# make install

Check that the configuration file /etc/ld.so.conf contains the line : 

/usr/local/lib
If it is not present, as root, add the line and then run ldconfig.

Install vlc

Download the sources of the latest release : get the file vlc-version.tar.gz from the vlc download page. Uncompress :

% tar xvzf vlc-version.tar.gz
% cd vlc-version

To get the list of configuration options, do : 

% ./configure --help
Configure vlc. For example :
  • if you want a basic vlc, do : 
    % ./configure
  • if you want the Gnome interface instead of the GTK interface (you will need the developpement packages of Gnome) 
    % ./configure --enable-gnome
  • if you want a full vlc with libdvbpsi, ffmpeg and mad (a52 and libdvdcss are enabled by default) : 
    % ./configure --enable-dvbpsi --enable-mad --enable-ffmpeg --with-ffmpeg-tree=/usr/local/src/ffmpeg-date
    if you uncompressed ffmpeg in the directory /usr/local/src/ffmpeg-date.

Then, compile and install : 

% make
% su
 [enter root password]
# make install

You can also do a make uninstall, make clean or make distclean as needed.

2.2 Run vlc

Read an MPEG file

Download a valid MPEG file, for example ftp://ftp.videolan.org/pub/videolan/streams/presentation/presentation_short.vob.

To make vlc read this file, type the following at the command line :

 
% vlc presentation_short.vob
To get more information on the possible command line arguments, do : 
% vlc --help

You can also use the interface : 

% vlc
and then click on "File" and select the file.

Read a DVD

If you are running Linux, you must have write access to the device corresponding to your DVD drive. For that, you should be in the disk or cdrom group (look at the permissions in /dev). As root, do :

# adduser my_login disk_or_cdrom
and then close your user session and open it again.

If you are running Windows 2000 or XP Pro, you should have administrator rights (i.e. logging in as administrator or as a user with administrator rights).

To make vlc read a DVD, you can tell him from the command line : 

% vlc dvd:/dev/dvd
where /dev/dvd is the device that corresponds to your DVD drive.

Under Windows, it becomes : 

% vlc dvd:D:\
where D: is your DVD drive.

You can also use the interface : 

% vlc
and then click on "DVD" and select your DVD drive (/dev/dvd under Unix, D:\ under Windows).

If vlc doesn't work

  1. First, launch vlc in verbose mode : 
    % vlc -vvv
    and try to understand the error message.
  2. Then, read vlc HOWTO and vlc FAQ (also available in the sources, file "FAQ"),
  3. If you can't find out the problem, explain your problem in English in the mailing-list vlc@videolan.org with the output of vlc -vvv.
